Первый коммит - сайт аналог Пиццы Фабрики и подобных заведений. 
Существует два раздела для user'ов и админа.
Первый раздел - клиентоориентированные, красивые две странички, первая с общей информацией, вторая - меню, сделано было исключительно без js, только html+css
Второй раздел - панель Админа, первый дизайн который я сделал, не стал менять на более красивый, т.к. он и так остается интуитивно понятным. В этой панели админа реализуется CRUD операции.
Можно создать/прочитать/изменить/удалить категории блюд, потом то же самое можно сделать с блюдами, с их свойствами и так далее.
На данный момент для связи с БД используется postgre sql, jdbcTemplate. Также на данный момент в Spring Security config пароль для роли админа не выставлен, это будет в заключительной готовой версии проекта.
